Introduction

In this portfolio, we decided to use our knowledge of HTML Canvas and JavaScript to create a "true" Snake game. A JavaScript library, p5.js was used to create the game along with Bootstrap to create this introduction page.

Instructions

To play, simply click the link at the right which will lead you to the game. Objective of the game is to control the snake (white square) to eat the food (red square). Snake gets larger with each food eaten. The instructions to play are as follows:

UP ARROW or W: move up

DOWN ARROW or S: move down

LEFT ARROW or A: move left

RIGHT ARROW or D: move right

Hitting the edge of the canvas or eating your tail will result in a death and you will have to start over!
